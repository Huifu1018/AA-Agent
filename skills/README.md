# Agent-S Skills

This directory stores reusable workflow skills for the local Agent-S project.

Suggested convention:

- One subdirectory per skill
- Each skill contains a `SKILL.md`
- Keep the skill focused on one workflow or capability

Current skills:

- `gmail_attachment_reply`: Gmail unread-mail handling with latest-body extraction, attachment-aware reasoning, reply-all sending, quoted replies, and mark-as-read behavior
